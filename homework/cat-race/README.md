# <h1> 고양이 달리기 프로젝트 </h1>

1. 드롭박스로 몇 마리가 뛸 지 결정한다.
2. 생성 버튼을 누르면 선택한 숫자대로 트랙과 고양이가 생성된다.
3. [출발] 버튼을 누르면 고양이가 달리기 시작한다. 어떤 고양이가 더 빠른지는 랜덤
4. 고양이가 전부 골인하면 순위와 시간이 표시된다.

<hr>

<h2>프로젝트 소개</h2>
JAVASCRIPT와 CSS를 이용한 프로젝트로, 선택한 고양이 수 만큼 고양이가 달리기를 하는 게임이댜.
